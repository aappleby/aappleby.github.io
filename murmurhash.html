MurmurHash is a hash function that I invented while trying to teach myself about cryptography.

It turned out to be faster than most existing 'fast' hash functions, so I published it. It's now on version 3, and the source code and some documentation here - http://code.google.com/p/smhasher/

The old MurmurHash site is here - http://sites.google.com/site/murmurhash/
